# WebNavigatorAI_TheOne
WebNavigatorAI, built by team "The One" for Bytexl Hacxlerate, is a local AI agent that executes natural language web tasks (e.g., "search laptops under 50k") using Ollama LLM and Playwright. Features GUI, voice input, JSON/CSV export, and memory. Fully offline, no cloud needed. #Hacxlerate
